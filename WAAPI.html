Web Animations API (WAAPI), announced in 2014, is a new and experimental technology, still being developed. It promises to combine the benefits of both CSS and JavaScript, allowing for synchronised and timing changes to the presentation of a webpage by combining timing and animation models.
WAAPI allows the browser's animation engine to be manipulated directly by JavaScript and with it we can move interactive animations from style sheets to JavaScript, therefore, seperating presentation from behaviour. This means even more control and flexibility than JavaScript, alone, provides, making it one of the most performant ways to animate on the web and with it still being in relatively early days of development, more animation features can and are likely to be added in the near future.
